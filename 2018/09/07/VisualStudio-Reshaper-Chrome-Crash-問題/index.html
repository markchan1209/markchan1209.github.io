VisualStudio Reshaper Chrome Crash 問題

發表於 2018-09-07 | 分類於 Issue | 閱讀次數:

原因

三個月前更新了Reshaper VS2017 Chrome 結果 每次寫好程式要用VS2017 DEBUG
網頁瀏覽器開起來 按個F12(Chrome開發者工具) 結果就會閃退(瞬間關閉DEUBG模式)

弄了老半天，CHROME也重灌了，結果還是一樣
查了文章
後來就把我的VS2017更新 CHROME更新，重開機，一切都好了~~~

分享給大家
