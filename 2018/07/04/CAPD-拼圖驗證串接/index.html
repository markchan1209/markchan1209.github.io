CAPY拼圖驗證 WEB + Andriod

發表於 2018-07-04 | 分類於 Andriod | 閱讀次數:

需求

公司需要做拼圖驗證，上面找了一家日本廠商CAPY的拼圖驗證平台做串接，成功時候要回傳Andriod的JS

CAPY拼圖驗證介紹

特色

  • 後台可以監控拼圖使用數據、設定拼圖模式、格式、難度。 (剩下請看官網)

CAPY 拼圖驗證流程

  • 第一、第二 步驟 進入WEB
  • 第三、第四 步驟 換拼圖驗證
  • 第五、第六、第七 步驟 驗證拼圖

串接方式

  • 進入CAPY廠商後台， 選擇 Puzzle CAPTCHA > +ADD 產生一組拼圖JS

  • 將JS貼到HTML頁面上，並且加上

  • 使用JS AJAX POST CAPY指定參數的到後端，

  • 後端再透過 SERVER TO SERVER Form Post 將KEY 跟 指定參數一起傳到CAPY做驗證。
  • 用SERVER TO SERVER是因為 瀏覽器有檔 JS CORS跨網域存取，而且把KEY放在SERVER端比較安全。
