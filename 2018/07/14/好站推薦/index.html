ZooKeeper

是一個分佈式的分層文件系統，它有助於客戶端之間的鬆散耦合，並提供其znode的最終一致視圖，這些視圖類似於傳統文件系統中的文件和目錄。它提供基本操作，如創建，刪除和檢查znode的存在。它提供了一個事件驅動模型，客戶端可以在其中監視特定znode的更改，例如，如果將新子項添加到現有znode中。ZooKeeper通過運行多個名為ensemble的 ZooKeeper服務器實現高可用性，每個服務器都持有分佈式文件系統的內存副本以服務客戶端讀取請求。

Cloud Datastore

Cloud Datastore 是具高度擴充性的 NoSQL 資料庫，適用於您的應用程式。Cloud Datastore 會自動處理資料分割及複製作業，也能隨著應用程式的負載而自動進行調整，因此穩定且可靠，方便您隨時存取。Cloud Datastore 的功能多樣齊全，包括 ACID 交易處理、類似 SQL 的查詢技術以及建立索引等功能。

CLOUD SPANNER

Cloud Spanner 是唯一專為雲端打造、遍及全球，且可維持高度一致性的企業級資料庫服務，其結合了關聯資料庫結構和非關聯水平資源配置的優點，因此可在所有資料列、區域和洲別中讓交易以高效能進行，同時提供高度一致性；此外，服務水準協議的可用性達 99.999%，在業界無出其右，使用者無需進行停機維護作業，還可享有企業級的安全性。Cloud Spanner 徹底改變了資料庫管理方式，並讓開發人員可更有效率地開發應用程式。

在當今分秒必爭的全球化世界中，IT 與開發人員的效率 (評估標準為應用程式停機時間和上市時間) 就是機構最寶貴的資源之一。如何有效管理應用程式資料庫後端，同時為開發人員提供必要工具以有效率地進行建置，這在過去曾經是棘手的難題。
